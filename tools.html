Business Tools

Tools used for this Assignment

Microsoft Teams

Microsoft Teams was used heavily throughout the project (Microsoft 2021). It has the chat and video meeting functionality required for all the communication and planning, as well as file storage capabilities required for staging and sharing content.

Comments

After the group was formed in Canvas, Microsoft Teams was agreed to be the most appropriate form of communication. Canvas was then no longer used with relation to the project, with MS Teams fulfilling almost all the group’s needs.

GitHub

A provider of internet hosting for software development and version control, GitHub (2021) was used with the building of the team’s website. The repository can be found here.

Comments

As all the files were staged through MS Teams, the repository’s audit trail will not be reflective of how the group’s input was shared. Only Karsten, Jessica and Alexander engaged with GitHub in any meaningful way to get the website active - all created content was shared through MS Teams. This means that, while the work was mostly distributed fairly throughout the group, the GitHub repository activity will look very lopsided. It was felt that, even though there were multiple contributors, the version control capabilities of Git would not be necessary, focusing our attention on one point of contact. This allowed us to have the person with the most experience and talent with web design, Karsten, handle the production of the website, with all content for it provided through MS Teams.

Bootstrap

Bootstrap (2021) is an open-source framework of CSS classes that makes the creation of mobile friendly and responsive websites quicker and easier. Bootstrap provides a stylesheet with predefined classes that speed up page layout and hence, the development process.

Comments

Bootstrap was used by Karsten in designing the website.

Adobe Dreamweaver

Dreamweaver, part of the Adobe Creative Cloud, is a WYSIWYG editor, that allowed for the website to be put together in a meaningful way before uploading (Adobe 2021). It includes Git functionality that allows for the changes to be staged and committed directly to GitHub.

Comments

Dreamweaver was used by Karsten in building the website.

Arduino IDE

Open-source Arduino software that enables the writing and uploading of code to Arduino and compatible boards (Arduino 2021).

Comments

The Arduino IDE was used to upload code to the Duinotech board for the Baby Boom Gate project.

Celtx Storyboarding

Cloud computing based solution for writing, planning, and managing media production ranging from film and television to video games (Celtx 2021).

Comments

Celtx is being used by Ashley and Alex for collaborative preproduction workflow for Assessment 5’s script and preliminary storyboard.

Microsoft PowerPoint

Microsoft PowerPoint is a slideshow-based presentation program that can be used to present information and pitches (Microsoft 2021).

Comments

Originally Celtx was going to be utilised for producing the final storyboard however it lacked the same level of presentation capability that was sought after for. Alex made the decision to go to PowerPoint as it was both the most familiar program and more well-presentable, aesthetically.

Adobe Premiere Pro

Adobe Premiere Pro is a timeline-based video editing software application part of the Adobe Creative Cloud licensing program (Adobe 2021).

Comments

Premiere Pro is being utilised by Ashley and Alex for Assessment 5 to import the footage recorded, edit, and then export that footage as synced and coherent video edits.

Avid Pro Tools

Pro Tools is a digital audio workstation that can record, edit, mix, and master audio for professional production (Avid 2021).

Comments

Pro Tools being used by Alex, will be for editing dialogue and the music, and sound effects for post-production production in the video edit.
