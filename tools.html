A2 Group 10 - Tools

Tools used for this Assignment

Microsoft Teams

Microsoft Teams was used heavily throughout the project. It has the chat and video meeting functionality required for all the communication and planning, as well as file storage capabilities required for staging and sharing content.

Comments

After the group was formed in Canvas, Microsoft Teams was agreed to be the most appropriate form of communication. Canvas was then no longer used with relation to the project, with MS Teams fulfilling almost all of the group’s needs.

GitHub

A provider of internet hosting for software development and version control, GitHub was used with the building of the team’s website. The repository can be found here.

Comments

As all the files were staged through MS Teams, the repository’s audit trail will not be reflective of how the group’s input was shared. Only Karsten, Jessica and Alexander engaged with GitHub in any meaningful way to actually get the website active - all created content was shared through MS Teams. This means that, while the work was mostly distributed fairly throughout the group, the GitHub repository activity will look very lopsided. It was felt that, even though there were multiple contributors, the version control capabilities of Git would not be necessary, focusing our attention on one point of contact. This allowed us to have the person with the most experience and talent with web design, Karsten, handle the production of the website, with all content for it provided through MS Teams.

Bootstrap

Bootstrap is an open-source framework of CSS classes that makes the creation of mobile friendly and responsive websites quicker and easier. Bootstrap provides a stylesheet with predefined classes that speed up page layout and hence, the development process.

Comments

Bootstrap was used by Karsten in designing the website.

Adobe Dreamweaver

Dreamweaver, part of the Adobe Creative Cloud, is a WYSIWYG editor, that allowed for the website to be put together in a meaningful way before uploading. It includes Git functionality that allows for the changes to be staged and committed directly to GitHub.

Comments

Dreamweaver was used by Karsten in building the website.

Arduino IDE

Open-source Arduino software that enables the writing and uploading of code to Arduino and compatible boards.

Comments

The Arduino IDE was used to upload code to the Duinotech board for the Baby Boom Gate project.
